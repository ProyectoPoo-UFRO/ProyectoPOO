# Proyecto POO – Simulador Máquina Expendedora

## **Equipo de desarollo**
- Javier Cortés (22284185225)
- Patricio Ulloa (21799721625)
- Carlos Ramirez (21995122125)

Este proyecto corresponde al desarrollo de un programa que simula el funcionamiento de una maaquina expendedora de bebidas y productos en general, proyecto realizado para la asignatura **Programación Orientada a Objetos (ICC703-1)**. El programa se construye utilizando una arquitectura **Frontend – Backend – Base de Datos**, incorporando tecnologías modernas como **React**, **Java Spring Boot** y **MongoDB**.

El objetivo principal es crear una página web interactiva, modular y escalable que permita administrar datos, realizar operaciones CRUD, visualizar mapas, manejar usuarios y más.
